Rec Filter Component behavior

  • Initial state all filters are closed. A + button is shown on the right of each filter
  • When a filter is open but there is no value or selection on that filter the '-' button shows to close that filter
  • When a selection is made or the filter has value then an 'X' button is shown to reset that filter
  • When a filter has a selection or value filter cannot be fully closed. It shows a widget with the value or selection.
  • When a clicking on a widget inside the filter that filter goes into edit mode.
  • When selecting a filter, all other filters get fully closed if they have no value or the filter shows the widget value
  • The component notifies the host that new values for the filter are in everytime any filter changes.
